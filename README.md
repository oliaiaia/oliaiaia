### RRT ‚Äî Rapid Random Tree Planner
**C++ ¬∑ Eigen ¬∑ Python ¬∑ pybind11**


A C++ library for nonlinear path planning in continuous space, based on the Rapidly-Exploring Random Tree (RRT) algorithm. It is designed for robotics and simulations that require consideration of kinematic constraints and complex obstacles. The library includes Python bindings for intuitive integration into Python scripts and projects (e.g., for ROS, simulators, or data analysis).

### aStartStick ‚Äî Discrete-Angle A* Planner
**C++ ¬∑ Eigen ¬∑ Python ¬∑ pybind11**


A C++ implementation of the classic A* algorithm for pathfinding in a discrete state space, where each state is described not only by coordinates (x, y) but also by a discrete orientation (Œ∏). This makes the algorithm applicable to robots with kinematic constraints (e.g., cars, forklifts), where turning in place is impossible or limited. The implementation focuses on efficient work with multidimensional state spaces.
